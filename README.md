# MCP Host Web [第四课]

把 [MCP Host](https://github.com/guobinqiu/mcp-host) (命令行版) 改成 Web 版

## 运行

1. 启动后端服务

```
cd backend && go run main.go
```

2. 启动前端服务

```
cd frontend && npm run serve
```
